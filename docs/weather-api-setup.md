# 기상청 API 연동 설정 가이드

## 1. 기상청 API 키 발급

### 공공데이터포털에서 API 키 발급받기
1. [공공데이터포털](https://www.data.go.kr/) 접속
2. 회원가입 및 로그인
3. "기상청_단기예보 ((구)동네예보) 조회서비스" 검색
4. 활용신청 → 일반인증키(Encoding) 선택
5. 발급받은 인증키를 환경 변수에 설정

### API 서비스 정보
- **서비스명**: 기상청_단기예보 ((구)동네예보) 조회서비스
- **제공기관**: 기상청
- **API 유형**: REST API
- **데이터 형식**: JSON, XML
- **호출 제한**: 일 1,000건

## 2. 환경 변수 설정

Vercel 프로젝트 설정에서 다음 환경 변수를 추가하세요:

**변수명**: `KMA_API_KEY`  
**값**: 발급받은 기상청 API 인증키

> 참고: 이 변수는 서버 사이드에서 기상 정보를 가져오기 위해 사용됩니다.

## 3. 동해시 기상 정보

현재 시스템은 동해시 기상 정보를 가져오도록 설정되어 있습니다:
- **격자 좌표**: X=102, Y=95
- **지역**: 강원도 동해시
- **업데이트 주기**: 10분마다 자동 갱신

## 4. 제공되는 기상 정보

- **기온** (T1H): 현재 기온 (°C)
- **습도** (REH): 상대습도 (%)
- **풍속** (WSD): 풍속 (m/s)
- **풍향** (VEC): 풍향 (도)
- **강수량** (RN1): 1시간 강수량 (mm)

## 5. 비상상황 활용

기상 정보는 다음과 같이 비상상황 대응에 활용됩니다:

### 바람 정보 활용
- **유해물질 누출 시**: 바람 방향을 고려한 대피 경로 안내
- **화재 발생 시**: 연기 확산 방향 예측

### 강수량 정보 활용
- **대피 시 주의사항**: 미끄럼 방지 안내
- **실내 대피소 우선 추천**: 강우 시 실내 시설 선호

### 시정 정보 활용
- **시야 불량 시**: 대형 건물 위주 대피소 추천
- **야간 대피**: 조명이 잘 갖춰진 장소 우선 안내

## 6. 오류 처리

API 호출 실패 시 시뮬레이션 데이터로 자동 전환되어 시스템 중단 없이 운영됩니다.
